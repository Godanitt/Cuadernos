# Diseños de portada

Formatos disponibles para `notebook.cover.style`:

- `solid`
- `fullimage` / `cover4`
- `wiley` / `wiley1` / `cover1`
- `wiley2` / `cover3`
- `fullimage2` / `cover5`
- `wiley2.1` / `cover6`
- `wiley2.2` / `cover7`

Esta carpeta contiene únicamente el código Typst de los diseños. Las imágenes reales de portada se guardan en `Imagenes/` dentro de cada cuaderno.
