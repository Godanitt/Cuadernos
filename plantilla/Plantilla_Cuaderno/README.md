# Módulos de la plantilla

Componentes internos usados por `../plantilla.typ`:

- `indice.typ` e `indice_aux.typ`: índices y sumarios.
- `titulo.typ`: configuración de portada y título.
- `theorems.typ`: entornos de teoremas.
- `comandos.typ`: comandos auxiliares.
- `portadas/`: diseños de portada e imágenes base.
