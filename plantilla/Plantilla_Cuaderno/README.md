# Módulos internos de la plantilla

Componentes utilizados por `../plantilla.typ`:

- `indice.typ` e `indice_aux.typ`: índices y sumarios.
- `titulo.typ`: selector de portada.
- `theorems.typ`: entornos matemáticos.
- `comandos.typ`: comandos auxiliares.
- `portadas/`: código Typst de los diseños de portada.

`portadas/` ya no contiene una biblioteca de imágenes. Las imágenes reales se encuentran en `Imagenes/` dentro de cada cuaderno.
